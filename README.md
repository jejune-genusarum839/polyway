## 🚀 How to Use the App

Launch the application using the new shortcut on your desktop. The main window shows a clean dashboard. You see your active tasks on the left and your file list on the right. 

To add a new task, click the plus icon at the top of the task pane. Type a name for your task and press the Enter key. You can drag and drop files from your computer into the file pane to categorize them. The app remembers these locations for your next session.

## 🛠️ Troubleshooting Issues

If the software fails to open, check these common fixes:

* Restart your computer. This clears background bottlenecks.
* Check your internet connection. The app performs a quick update check during startup.
* Run the installer again if the desktop shortcut does not appear.
* Use the Windows Search bar to look for "polyway" if you cannot find the icon.

Contact your system administrator if you see a permission error. This error happens if your user account lacks rights to install new software. Most home computers allow this by default.
